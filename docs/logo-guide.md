# Logo Guide

## Structure

Each product uses the same icon system with a different wordmark:

- **Icon:** Open book with a cross symbol above it
- **Colors:** Sage green `#46725D` for the book icon; Amber `#C5843A` for the cross — this is the only place Amber appears in the logo
- **Background:** Sage Light `#E6EFEB` rounded square container

## Usage Rules

- Never recolor the logo. Sage and Amber only.
- Always use the transparent version (`icon-transparent.png`) on colored backgrounds.
- Minimum clear space: equal to the height of the cross symbol on all four sides.
- Never stretch, rotate, skew, or add effects.
- Never place the logo on a dark background unless using an approved dark variant.

## Product Wordmarks

| Product | Wordmark text |
|---|---|
| Parent brand | `myScripture` |
| Notes | `myScripture Notes` |
| Journey | `myScripture Journey` |
| (all others follow the same pattern) | `myScripture [Product]` |

## File Locations (in production apps)

- `/logos/notes/icon-transparent.png`
- `/logos/journey/icon-transparent.png`
- `/public/favicon.ico`, `/public/icon.png`
