# myScripture Design System

A complete brand and design system reference for the [myScripture](https://myscripture.app) ecosystem.

**Primary use:** Feed `CLAUDE.md` into Claude Design (or any AI tool) to generate new myScripture apps that are visually and tonally consistent with the existing product family.

---

## Quick Start for AI Tools

Paste the contents of [`CLAUDE.md`](./CLAUDE.md) at the top of your AI chat or project context. That file contains the complete brand quick-reference: colors, typography, spacing, component rules, product list, and guardrails.

## What's in This Repo

| Folder | Contents |
|---|---|
| [`tokens/`](./tokens/) | CSS variables + JSON for all design tokens |
| [`components/`](./components/) | HTML previews + React TSX for every UI component |
| [`products/`](./products/) | One styled HTML page per myScripture product (all 13) |
| [`docs/`](./docs/) | Visual guides: color palette, typography, spacing, voice |

## Brand in One Line

Sage green `#46725D` on ivory `#F6EFE5`. Lora headings, DM Sans body. Calm, warm, faithful.

## Products

13 products from available to future. See [`products/`](./products/) for individual pages, or [`CLAUDE.md`](./CLAUDE.md) for the full table.
