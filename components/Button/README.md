# Button

Always pill-shaped (`rounded-full`). Four variants.

| Variant | Use |
|---|---|
| `primary` | One per screen. Main CTA. Sage background. |
| `secondary` | Secondary actions alongside a primary button. |
| `ghost` | Tertiary, low-emphasis. Text links in UI. |
| `destructive` | Irreversible actions only (delete, remove). |

Never use amber or navy as button backgrounds. Sage is the only primary CTA color.
