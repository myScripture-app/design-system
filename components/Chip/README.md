# Chip

Pill-shaped label. Four color variants.

| Variant | Background | Use |
|---|---|---|
| `sage` | Sage Light | Active, available, primary category |
| `amber` | Warm Light | In-progress, warm category, prayer |
| `navy` | Navy Light | Tags, topics, tertiary category |
| `muted` | Stone | Future, inactive, neutral |

Add `dot={true}` for status indicators with a leading colored dot.
