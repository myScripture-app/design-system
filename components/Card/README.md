# Card

The primary surface element. Used for products, notes, content sections.

- Background: `--card` (rgb 253,253,252) on ivory `--background`
- Border: `1px solid --border-color`
- Radius: `16px` (rounded-2xl)
- Shadow: warm card shadow — never cold grey
- Hover: `translateY(-2px)` + heavier shadow, 0.22s ease

**Empty state cards** use a dashed border instead of solid.
