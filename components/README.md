# Components

Six UI components. Each folder contains:
- `preview.html` — open in any browser, zero setup required
- `Component.tsx` — React TSX for use in production apps (requires Tailwind)
- `README.md` — usage rules

| Component | Description |
|---|---|
| `Typography/` | Full type specimen — Lora headings, DM Sans body |
| `Button/` | Primary, secondary, ghost, destructive |
| `Card/` | Product card, note card, empty state |
| `Chip/` | Status badges and topic tags |
| `Input/` | Text, textarea, select, checkbox |
| `Navbar/` | Desktop + mobile responsive header |
